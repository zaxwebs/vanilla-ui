UI/004 Spacing

Spacing made Atomic

Inset, stacking - padding, margin.

Paragraph.

Direct text within column.

This project aims to implement spacing in the from of insets as paddings and stacking & inlines as margins. Space between parent and child is created using inset on parent. Vertical space between (generally after) children is defined with stacking while space while horizontal space between (generally after) children is created using inlines.

Spacing can be atomic with respective utility classes, which can offer higher modularity. If you don't prefer to go all-in on atomic css (especially when vanilla & at this scale), you can utilize the atomic spacing utilities only when needed such as when one child needs to be full width and the rest within inset.
