UI/007 Responsive Spacing

Static Mobile First Insets with .inset-{size}

.inset-1

.inset-1 has a fixed padding of 5px. The default scale follows multiples of 5, thus, inset-10 has a static padding of 10px and so on.

Responsive Insets with .inset-auto-{size}

.inset-auto-1

.inset-auto-{size} where padding is {size} * 5px below set breakpoint(s), for this demo, I've used only one breakpoint of 350px width and padding is {size} * 10px above set breakpoint(s).

Notes

  • This documentation specifies insets but the same mechanic follows for stacks and inlines.

  • .stack-auto-{3} can be seen in action on the above blue divs.

  • The choice of a scale that's multiples of 5 & 10 was arbitrary. Non-linear scales can be adopted as well. {size} can be none, a, b, c... for non-linear scales.
